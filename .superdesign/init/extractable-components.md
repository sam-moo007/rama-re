# Extractable Reusable Components

The following components from RAMA 2.0 can be extracted or converted to Superdesign canvas DraftComponents:

1. **`AppHeader`** (`apps/web/src/components/app-header.tsx`): Shared sticky navigation header with locale toggle and auth actions.
2. **`TrustPassport`** (`apps/web/src/features/property/components/trust-passport.tsx`): Bitemporal evidence passport component displaying claims, verification statuses, and provenance source tiers.
3. **`PropertyHero`** (`apps/web/src/features/property/components/property-hero.tsx`): Property media gallery, price summary, and key specifications.
4. **`CompareTable`** (`apps/web/src/features/compare/compare-table.tsx`): Side-by-side property comparison matrix with fit scores and tabular AED formatting.
